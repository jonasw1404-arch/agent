Mission Erklärung

Deine Aufgabe ist es, das Geheimnis des Hintergrunds zu entdecken und den Code zu knacken.

Auftrag

  1. Hinweise sammeln
  2. Rätsel lösen
  3. Code entschlüsseln
  4. Mission bestätigen
  5. Bericht abgeben

Rätsel 1: Geheimzahl

Finde die 4-stellige Zahl, die das Rätsel löst.

Rätsel 2: Code-Knacker

Gib die richtige 4-stellige Kombination ein.

Rätsel 3: Zahlen-Geheimnis

Welche 4 Ziffern passen hier?

Rätsel 4: Geheimcode

Tippe die richtige 4-stellige Kombination ein.

Rätsel 5: Der finale Code

Finde die letzte 4-stellige Kombination.
